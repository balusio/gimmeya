GimmeYa a delivery app
===========================
restaurant app based on API with restaurants lists :D 

## REQUIREMENTS:
* NODE (v10 LTS)
* DOCKER (2.1 for MAC)


Gimme ya is separated in 2 parts
1.- Front end React
2.- Node Express API with Redis


## DOCKER INSTANCE 

docker run --name redis_api -p 6379:6379 -d redis
